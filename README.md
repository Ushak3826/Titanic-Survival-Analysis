# Titanic Survival Analysis

** Objective:** 
Analyze Titanic dataset to find factors affecting passenger survival.

**Tools Used:** 
Python (Pandas, Seaborn, Matplotlib), 
SQL (SQLite), 
Jupyter Notebook  

**Data Cleaning:**
Filled missing Age and Embarked, 
dropped unnecessary columns, 
encoded Sex and Embarked  

**Key Findings:** 
Female survival ≈ 74%, 
Male ≈ 19%, 
Female 1st class highest survival  

**Conclusion:** 
Gender and passenger class were the most important factors for survival.
